# M-Market Electronics — Frontend

E-commerce storefront for electronics built with HTML, CSS, and Vanilla JavaScript.

- **Backend API:** [https://m-market-electronics.vercel.app/](https://m-market-electronics.vercel.app/)

## Features
- Browse & search products
- Filter by category (Phones, Laptops, Tablets, Watches)
- Product detail modal with order placement
- User authentication (Login / Register / Logout)
- Payment screenshot upload

## Setup
```bash
git clone https://github.com/mumekonin/m-market-electronics.git
cd m-market-electronics
# Open index.html with Live Server
```

## Deployment
Hosted on **Vercel** as a static site. Push to GitHub and import the repo on Vercel — no build command needed.
